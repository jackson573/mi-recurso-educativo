Día 4: Proyecto Tecnoeducativo en Robótica

Reto: Construcción de un Carro Robótico con Bluetooth

Introducción

En el último día del curso, se propone un reto práctico para aplicar y consolidar los conocimientos adquiridos. El objetivo es construir y programar un carro robótico controlado por Bluetooth, utilizando un microcontrolador Arduino y un módulo Bluetooth (HC-05 o HC-06). Este proyecto integrará áreas como electrónica, programación y diseño mecánico.

Además de reforzar habilidades técnicas, este reto fomenta la resolución de problemas, el pensamiento computacional y la creatividad. También ofrece a los docentes una guía para replicar esta actividad en el aula, promoviendo un aprendizaje significativo basado en la práctica.

Fundamentación Teórica

Carro Robótico:

Integra componentes como motores, controladores y módulos de comunicación.
Refuerza habilidades de pensamiento lógico y técnicas de programación.

Módulo Bluetooth:

Proporciona comunicación inalámbrica entre el Arduino y un dispositivo móvil.
Facilita el control remoto a través de comandos enviados desde una aplicación.

Secuencias Lógicas:

  • Diseño de algoritmos que controlen las acciones del carro, como avanzar, retroceder, girar y detenerse.
  • Introducción al pensamiento computacional y a conceptos de automatización.

Pasos para el Proyecto

Paso 1: Ensamble del Carro Robótico

  1. Diseño del chasis:
    • Utiliza materiales como madera, plástico o LEGO.
    • Diseña una estructura resistente para soportar los componentes electrónicos y los motores.
  2. Fijación de motores y ruedas:
    • Asegura los motores DC al chasis y conecta las ruedas.
    • Verifica que las ruedas estén alineadas para garantizar un movimiento estable.
  3. Estabilidad adicional:
    • Agrega una rueda loca (rueda libre) en la parte frontal o trasera para mayor estabilidad.

Pasos para el Proyecto

Paso 2: Integración del Circuito

  1. Controlador de motores (L293D):
    • Coloca el controlador en una protoboard.
    • Conecta sus entradas a los pines digitales del Arduino (ejemplo: Input 1 al pin 7, Input 2 al pin 8).
    • Asocia las salidas a los motores DC.
  2. Módulo Bluetooth (HC-05/HC-06):
    • Conecta los pines RX y TX del módulo a los pines digitales del Arduino (por ejemplo: RX al pin 10, TX al pin 11).
    • Asegúrate de cruzar las conexiones (RX a TX y TX a RX).
  3. Alimentación:
    • Utiliza una batería de 9V para los motores y el Arduino.
    • Revisa la polaridad para evitar daños a los componentes.

Pasos para el Proyecto

Paso 3: Programación del Control Bluetooth

Crea un programa en Arduino que interprete comandos enviados desde una aplicación Bluetooth. Por ejemplo:

Comandos:
F: Avanzar.
B: Retroceder.
L: Girar a la izquierda.
R: Girar a la derecha.
S: Detenerse.

Ejemplo de código:

Prueba el código enviando comandos desde una aplicación Bluetooth (por ejemplo, "Serial Bluetooth Terminal").

Actividad

Construcción y Programación del Carro Robótico

Objetivo: Construir un carro robótico funcional controlado por Bluetooth y programar su desplazamiento siguiendo secuencias lógicas que integren comandos avanzados.

Instrucciones:

  1. Ensambla el chasis:
    • Diseña un chasis utilizando materiales como madera, plástico o piezas prefabricadas. Asegúrate de que el diseño sea lo suficientemente resistente para soportar los motores y componentes electrónicos.
    • Fija los motores DC de manera segura y conecta las ruedas. Verifica que las ruedas estén alineadas para garantizar un movimiento estable y fluido.
    • Instala una rueda loca o un soporte adicional para mejorar la estabilidad del carro.
  2. Integra los componentes electrónicos:
    • Monta el controlador de motores L293D en una protoboard y conecta los motores siguiendo las instrucciones previamente estudiadas.
    • Conecta el módulo Bluetooth HC-05 a los pines digitales del microcontrolador Arduino. Asegúrate de cruzar las conexiones entre los pines RX/TX y TX/RX.
    • Alimenta el circuito utilizando una batería de 9V para los motores y el pin de 5V del Arduino para el controlador y el módulo Bluetooth.
  3. Programa el carro:
    • Abre el entorno de programación Arduino IDE y escribe un código que permita interpretar comandos enviados desde una aplicación Bluetooth.
    • Por ejemplo:
      • Comando ‘F’: Avanzar.
      • Comando ‘B’: Retroceder.
      • Comando ‘L’: Girar a la izquierda.
      • Comando ‘R’: Girar a la derecha.
      • Comando ‘S’: Detener.
    • Asegúrate de incluir un sistema que permita ajustar la velocidad mediante señales PWM.
  4. Prueba el carro:
    • Descarga una aplicación de control Bluetooth en tu dispositivo móvil (por ejemplo, "Serial Bluetooth Terminal").
    • Empareja el módulo Bluetooth con tu dispositivo y prueba enviando comandos desde la aplicación.
    • Observa el comportamiento del carro y realiza ajustes en el código si es necesario.
  5. Diseña una secuencia lógica:
    • Crea una secuencia lógica que permita al carro realizar una ruta predefinida.
      • Por ejemplo:
        • Avanzar por 3 segundos.
        • Girar a la derecha durante 1 segundo.
        • Retroceder por 2 segundos.
    • Implementa esta lógica en el código y prueba el desplazamiento del carro.
  6. Documenta el proceso:
    • Toma fotos del ensamblaje del carro y de los esquemas de conexión.
    • Graba un video del carro ejecutando la secuencia lógica.
